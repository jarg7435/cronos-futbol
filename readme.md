# ⚽ Cronos Fútbol - Coach Assistant

**Cronos Fútbol** es una aplicación web progresiva (PWA) diseñada para entrenadores de fútbol (F7 y F11) que necesitan llevar un control exhaustivo del tiempo de juego de sus jugadores, gestionar cambios y generar reportes detallados.

🔗 **[Abrir Aplicación (Online)](https://jarg7435.github.io/cronos-futbol/)**

## 🚀 Características Principales

*   **⏱️ Control de Tiempo Real:** Cronómetro global y temporizadores individuales por jugador.
*   **👥 Gestión Completa de Equipos:**
    *   Soporte para **Local** y **Visitante**.
    *   Banco de suplentes independiente.
    *   Personalización total (Nombre, Colores de Camiseta/Pantalón/Número).
*   **🔄 Cambios Inteligentes:**
    *   Sistema *Drag & Drop* (Arrastrar y Soltar) para sustituciones.
    *   Detección automática de titular/suplente.
*   **📊 Marcador Interactivo:**
    *   Clic en el marcador para editar goles manualmente.
*   **💾 Persistencia de Datos:**
    *   Guarda tus alineaciones y equipos frecuentes.
    *   Carga rápida desde el menú de configuración.
*   **📥 Reportes Profesionales:**
    *   Exporta los datos del partido a **CSV compatible con Excel**.
    *   Incluye: Resultado final, Tiempos de entrada/salida detallados y Tiempo total jugado.
*   **📱 Diseño Responsivo (PWA):**
    *   Optimizado para **iPad y Móviles** en modo horizontal (Landscape).
    *   Instalable como App Nativa (sin barras de navegador).

## 🛠️ Cómo Usar

1.  **Configuración Inicial:**
    *   Al abrir, configura los nombres y colores de los equipos **Local** y **Visitante**.
    *   Elige el modo: **Fútbol 7** o **Fútbol 11**.
    *   (Opcional) Carga equipos guardados previamente.

2.  **El Partido:**
    *   Pulsa **EMPEZAR** para iniciar el cronómetro.
    *   **Sustituciones:** Arrastra un jugador del banquillo al campo (o sobre otro jugador para cambio directo).
    *   **Goles:** Toca el número en el marcador para sumar goles.

3.  **Finalizar:**
    *   Pulsa **DESCARGAR** para obtener el informe completo del partido.
    *   Pulsa **GUARDAR** para almacenar la configuración de los equipos para el futuro.

## 📦 Instalación (Móvil/Tablet)

1.  Abre el enlace en Safari (iOS) o Chrome (Android).
2.  Toca el botón **Compartir** o **Menú**.
3.  Selecciona **"Añadir a Pantalla de Inicio"**.
4.  ¡Listo! Ahora funcionará como una app a pantalla completa.
